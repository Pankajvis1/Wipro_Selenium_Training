# Wipro Selenium Training

## 📌 About

This repository contains the projects, assignments, and practice
implementations completed as part of my **Wipro SDET (Software
Development Engineer in Test) Training Program**.

It serves as a collection of hands-on exercises and mini-projects
designed to strengthen my understanding of software testing, automation
frameworks, and Core Java programming concepts while following
industry-standard practices.

## 📚 Topics Covered

Throughout the training, the projects and assignments cover:

-   Core Java
-   Object-Oriented Programming (OOPs)
-   Exception Handling
-   Collections Framework
-   File Handling
-   Java 8 Features
-   Selenium WebDriver
-   TestNG
-   Maven
-   Page Object Model (POM)
-   Data-Driven Testing
-   Cross-Browser Testing
-   Explicit & Implicit Waits
-   Assertions and Test Validation
-   WebDriverManager
-   Logging (Log4j/SLF4J)
-   Reporting (Extent Reports)
-   Jenkins CI/CD
-   Docker Integration
-   Git & GitHub
-   Automation Framework Development
-   Test Automation Best Practices
